# Holding-Pattern

Air Routs — компьютерная стратегическая игра в жанре головоломки, в которой игрок может построить эффективные авиалинии, разработанная с использованием Godot Engine под операционную систему Аврора в качестве учебного проекта.

Особенности игры

    Каждый аэропорт находится в некоторой точке и изображается на карте одной из геометрических фигур (квадрат, треугольник, звезда и тому подобное). Фигура определяет тип пассажиров, которых аэропорт может принять. Со временем на точках появляются люди, изображаемые в виде геометрических фигур меньшего размера; их форма соответствует тем аэропортам на которые пассажиры хотят приехать. Пассажиры появляются на станциях случайным образом.

Игровые механики

    Игрок может строить авиалинии, связывающие аэропорты, посредством рисования маршрутов в виде дуг. По построенным маршрутам начинают летать самолеты — они останавливаются на точках, где происходят посадка и высадка пассажиров.Помимо этого, игроку доступен ряд других манипуляций с игровыми объектами. Так, игрок может перебрасывать самолеты с одной линии на другую, добавлять новые самолеты по ходу игры. Появление со временем новых аэропортов и возрастающий поток пассажиров приводят к тому, что транспортная сеть становится более сложной в управлении, в чём и заключается игровой вызов.
